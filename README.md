# 🍔 Swiggy Clone - Food Management Platform

A professional **Full-Stack Food Management Application** built using the Decoupled Architecture. This project features a modern React frontend and a robust Java Spring Boot backend to manage foods, restaurants, and user roles efficiently.

---

## 🚀 Features

- **Dynamic Dashboard:** Role-based access for Admin, Customers, and Delivery Partners.
- **Management System:** 
  - Add and Manage Food Items with real-time updates.
  - Manage Restaurant listings and categories.
  - User Management system with secure role assignment.
- **Modern UI/UX:** 
  - Smooth **Syne** and **DM Sans** typography.
  - Interactive input fields with focus glow effects.
  - Fully responsive navigation and layout.
- **Secure Authentication:** Protected routes that require sign-in to access dashboard features.

---

## 🛠️ Tech Stack

**Frontend:**
- React.js (Vite)
- CSS3 (Custom variables & Flexbox)
- React Router DOM (Navigation)
- Axios (API Integration)

**Backend (Decoupled):**
- Java Spring Boot
- MySQL Database
- RESTful APIs

---

## 📸 Interface Preview

- **Authentication Page:** Modern login/register toggle with "Jump to Dashboard" quick links.
- **Admin Panel:** Specialized view for managing the database entries.
